(Hartshorne II.7.7)

Let \(s \in \Gamma(X, \L(D))\) and suppose that corresponds to a rational function \(f\) on \(X\text{.}\) Suppose \(D\) is locally given by \(\divisor(g)\) on some open set \(U\text{.}\) Then the isomorphism \(\L(D) \to \O_X\) takes \(s \mapsto fg\text{.}\) In other words, \(\divisor(s) = D + \divisor(f)\) on every open set. Thus \(\divisor(s) \sim D\) via the rational function \(f\text{.}\) Additionally, \(s\) is a global section so it can't have any poles. Thus \(\divisor(s) \ge 0\text{.}\)

Conversely, suppose \(E = D + \divisor(f)\) with \(f \in k(X)\text{.}\) Then define a section \(s \in \L(D)\) which is locally \(f/g\) with the notation as above.

Finally, note that if \(\divisor(s) = \divisor(s')\) then \(\divisor(s/s') = 0\) and hence \(s/s' \in k\) since \(X\) is proper/projective.
