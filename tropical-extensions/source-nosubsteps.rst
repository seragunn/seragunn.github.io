Idylls
======

An *idyll* :math:`B = (B^\bullet, N_B)` is

* a monoid-with-zero :math:`B^\bullet = (B^\bullet, 0_B, 1_B, \cdot_B)`

* a proper ideal :math:`N_B \subset \N[B^\bullet]`

Such that :math:`B` is *field-like*

* :math:`0_B \neq 1_B`

* :math:`B^\bullet` is a group-with-zero

* there exists a unique :math:`-1` with :math:`(-1)^2 = 1 \text{ and } 1 + (-1) \in N_B`

Here

.. math:: \N[B^\bullet] = \{\text{finitely supported formal sums} \sum x_i\} / \langle 0_B \rangle

Alternative notation:

.. math:: \sum x_i \in N_B \longleftrightarrow 0 \leqslant \sum x_i

----

Ordered Blueprints
==================

A *subaddition* :math:`\leqslant` is an additive and multiplicative *preorder* on :math:`\N[B^\bullet]`:

* Reflexive: :math:`x \leqslant x`

* Transitive: :math:`x \leqslant y \wedge y \leqslant z \implies x \leqslant z`

* Additive and multiplicative: :math:`u \leqslant v \wedge x \leqslant y \implies (u + x \leqslant v + y) \wedge (ux \leqslant vy)`

* Antisymmetric on :math:`B^\bullet` (not on :math:`\N[B^\bullet]`)

----

Idyllic Blueprints
==================

:math:`B` is *idyllic* if :math:`\leqslant` is generated by relations of the form :math:`0 \leqslant x`

An idyll is a field-like idyllic blueprint

----

Examples
========

:math:`\F_{1^n}` has :math:`\F_{1^n}^\bullet = \{e^{2\pi i k/n} : k \in \Z\}` and :math:`N_{\F_{1^n}} = \{\sum \theta_i : \text{sum is } 0 \in \C\}`

:math:`\S` (sign idyll) has :math:`\S^\bullet = \{0, 1, -1\}` and :math:`N_{\S} = \{a \cdot 1 + b \cdot (-1) : ab \neq 0\}`

:math:`\T` (tropical idyll) has :math:`\T^\bullet = (\R \cup \{\infty\}, 0_\T = \infty, 1_\T = 0_\R, +)` and :math:`\N_{\T} = \{\sum x_i : \text{min. occurs twice}\}`

:math:`\K` (Krasner idyll) is the subidyll of :math:`\T` on :math:`\K^\bullet = \{0_\T, 1_\T\}`

----

Polynomial Extensions
=====================

:math:`B[x]^\bullet = \{ bx^n : b \in B^\bullet, n \in \N \}`

Whose subaddition is induced by that of :math:`B`

* Free :math:`B`-algebra

* :math:`\sum b_ix^{n_i} \leqslant \sum c_ix^{m_i}` if and only if the relation holds in each degree

A *polynomial* is :math:`\sum b_ix^i` (at most one term in each degree)

----

Tropical Extensions
===================

Let :math:`\Gamma` be an ordered Abelian group (e.g. :math:`\R`)

:math:`B[\Gamma]^\bullet = \{ bx^\gamma : b \in B^\bullet, \gamma \in \Gamma \}`

Where :math:`0 \leqslant \sum a_ix^{\gamma_i}` if and only if :math:`0 \leqslant \sum_I a_ix^{\gamma_i}` where :math:`I = \{\text{min. terms}\}`

(A similar construction appears in Bowler and Su's work on classification of stringent hyperfields)

----

More Generally
==============

We have an exact sequence of multiplicative groups:

.. math:: 1 \to B^\times \xrightarrow{\iota^\bullet} B[\Gamma]^\times \xrightarrow{v^\bullet} \Gamma \to 1

Coming from morphisms of idylls

.. math:: B \xrightarrow{\iota} B[\Gamma] \xrightarrow{v} \Gamma^{\rm idyll}

Such that exactness for groups: :math:`\im(\iota^\bullet) = \eq(v^\bullet, 1)` extends to exactness for idylls: :math:`\im(\iota) = \eq(v, 1)`

----

Examples
========

:math:`1 \to \K^\times \to \T^\times \to \R \to 1`

:math:`1 \to \K^\times \to \Gamma \to \Gamma \to 1`

:math:`1 \to \T_m^\times \to \T_{m + n}^\times \to \R^n \to 1` (where :math:`\T_m = \K[\R^m] = (\R^m, \le_{\rm lex})^{\rm idyll}`)

:math:`1 \to \S^\times \to \TR^\times \to \R^\times \to 1` (tropical reals)

:math:`1 \to \P^\times \to \TC^\times \to \R^\times \to 1` (tropical complexes, phase idyll)

----

Multiplicities
==============

A factorization is :math:`0 \leqslant f(x) - (x - a)g(x)`

The multiplicity of :math:`f` at :math:`a` is :math:`0` if :math:`f` doesn't factor

and otherwise :math:`\mult^B_a f = 1 + \max_g \mult^B_a g` over all factorizations

(Definition comes from the work of Baker and Lorscheid on multiplicities over hyperfields)

----

Morphisms and Multiplicities
============================

* Morphisms preserve factorizations

* If :math:`(x - a)` can be factored out :math:`k`-times then the same is true after morphism

* Therefore :math:`\mult^B_a f \le \mult^C_{\varphi(a)} \varphi(f)` for any morphism :math:`\varphi : B \to C`

* Also, initial forms are morphisms so :math:`\mult^{B[\Gamma]}_{at^\gamma} f \le \mult^B_a (\In_\gamma f)`

----

Main Results
============

Factorizations of initial forms can be lifted

Implies :math:`\mult^{B[\Gamma]}_{at^\gamma} f = \mult^B_a (\In_\gamma f)`

----

Example 1
=========

:math:`f = 2 + 1x + 0x^2 + 0x^3 + 2x^4 + 1x^5 \in \T[x]`

.. image:: figure-1.svg
    :width: 500px

Initial forms: :math:`\In_1 f = 0 + x + x^2, \In_0 f = x^2 + x^3, \In_{-\frac12} f = x^3 + x^5`

Multiplicities: :math:`\mult^\T_1 f = 2, \mult^\T_0 f = 1, \mult^\T_{-\frac12} f = 2`.

----

Example 2
=========

Catalan OGF is a solution to :math:`C = 1 + tC^2`

Consider :math:`f(x) = +t^0 - t^0 x + t^1 x^2 \in \TR[x]`

Initial forms: :math:`\In_0 f = 1 - x, \In_{-1} f = -x + x^2 = x(x - 1)`

Conclusion: one positive root with valuation :math:`0` and one positive root with valuation :math:`-1`

.. math:: C_1 = 1 + t + 2t^2 + 5t^3 + \cdots, C_2 = \frac1t - 1 - t - 2t^2 - \cdots

----

Degree Bound
============

If :math:`\sum_b \mult^B_b f` for all polynomials in :math:`B[x]` then the same is true for any tropical extension of :math:`B`

Conclusion: degree bound holds for :math:`\text{fields}, \K, \S` and tropical extensions by these base idylls
